## Toolkit for building native desktop apps

Write your app in native markup and TypeScript — or Zig. The toolkit's own engine renders it into real OS windows — no browser, no WebView, no JS runtime.

For agents:
```bash
npx skills add vercel-labs/native
```

For human: 
``` bash
npm install -g @native-sdk/cli
```
